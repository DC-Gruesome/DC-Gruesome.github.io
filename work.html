The Iron Yard: Final Project - PokéDex

My final project for the Iron Yard is an interactive PokéDex style application powered by ReactJS. Users are able to select any Pokémon from the Pokémon world and review its basic stats, which are retrieved from the Pokemon API. Users may also select any two Pokémon to simulate how a basic fight between two wild Pokémon could play out. Though ready for Demo Day this project is far from over.
